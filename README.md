## Installation

1. Clone the repository locally.

```sh
git clone https://github.com/bunboop/vrc-auto-inviter.git \
cd vrc-auto-inviter
```

2. Build the project.

```sh
cargo build --release
```

3. Load VRChat cookies and group into the config file.

4. Run the project.

```sh
./target/release/vrc-auto-inviter
```
